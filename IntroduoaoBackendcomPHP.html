💻 Aula: Introdução ao Back-end com PHP

🎯 Objetivos da Aula

Ao final desta aula, o aluno será capaz de:

  • Entender o que é Back-end e Front-end
  • Saber onde entra o PHP nesse contexto
  • Comparar o PHP com o JavaScript
  • Fazer uma primeira visão prática de como o PHP funciona

🧩 1. O que é Front-end?

O Front-end é a parte do sistema que o usuário vê e interage — a “cara” do site ou aplicativo.
👉 É tudo o que está na tela do navegador.

Exemplos de tecnologias Front-end:

  • HTML: estrutura do site
  • CSS: aparência, cores e estilos
  • JavaScript: interatividade e lógica no navegador

Exemplo prático:
 Quando você clica num botão, preenche um formulário, vê uma animação — tudo isso é Front-end.

⚙️ 2. O que é Back-end?

O Back-end é a parte invisível do sistema — onde ficam as regras, os dados e a comunicação com o banco de dados.
👉 É o “cérebro” que processa o que o usuário faz no Front-end.

Responsabilidades do Back-end:

  • Receber dados do usuário
  • Processar informações
  • Acessar o banco de dados
  • Retornar resultados ao Front-end

Exemplo:
 Quando você faz login, o Back-end verifica seu e-mail e senha no banco de dados.

🐘 4. O que é PHP?

PHP significa “Hypertext Preprocessor”.
É uma linguagem de programação para o lado do servidor (Back-end), usada principalmente na web.

✨ Características principais:

  • Facilita a criação de páginas dinâmicas (que mudam de acordo com o usuário).
  • Muito usada em sites como WordPress, Facebook e Wikipédia.
  • Funciona junto com HTML — você pode colocar código PHP dentro de uma página HTML.

⚡ 5. PHP na prática (exemplo simples)

<!DOCTYPE html>

<html>

<body>

  <h1>Exemplo com PHP</h1>

  <?php

    $nome = "Maria";

    echo "Olá, $nome! Seja bem-vinda ao nosso site!";

  ?>

</body>

</html>

🔹 O PHP roda no servidor, gera o HTML e envia para o navegador.
🔹 O navegador recebe apenas o resultado, não vê o código PHP.

🧩 Exemplo comparando PHP e JavaScript

JavaScript (lado do cliente):

<script>

  let nome = prompt("Digite seu nome:");

  alert("Olá, " + nome + "!");

</script>

PHP (lado do servidor):

<?php

  $nome = "Maria";

  echo "Olá, $nome!";

?>

🔹 O JavaScript mostra a mensagem no navegador.
🔹 O PHP gera a mensagem antes de a página ser carregada.

💡 7. Como o PHP se conecta com o Front-end

Fluxo típico:

  1. O usuário preenche um formulário em HTML
  2. O navegador envia os dados ao PHP (Back-end)
  3. O PHP processa e, se quiser, salva no banco de dados
  4. O PHP responde ao usuário com uma nova página ou mensagem

🚀 8. Mini Desafio Prático

Crie um arquivo chamado teste.php com o seguinte código:

<?php

  $nome = $_GET['nome'];

  echo "Olá, $nome! Este texto veio do servidor!";

?>

Depois, abra no navegador:
👉 http://localhost/teste.php?nome=João

O servidor PHP vai responder com:

Olá, João! Este texto veio do servidor!

🧭 9. Conclusão

  • Front-end: o que o usuário vê e interage
  • Back-end: onde tudo acontece por trás
  • PHP: uma das linguagens mais usadas no Back-end
  • JavaScript: nasceu para o Front-end, mas também pode fazer Back-end (Node.js)

👉 Juntos, Front-end + Back-end formam um sistema completo.

🧩 1. Primeiro programa em PHP

<?php

// Comentário de uma linha (como // em JavaScript)

// Exibe uma mensagem na tela

echo "Olá, mundo!";

// Quebra de linha

echo "\nEste é meu primeiro programa em PHP!";

?>

🔍 Explicação linha a linha

  • <?php → indica o início do código PHP
  • // → comentário de uma linha (igual ao JavaScript)
  • echo → mostra algo na tela (equivalente a console.log() no JS)
  • \n → pula uma linha no terminal (como \\n em JS)
  • ?> → marca o fim do bloco PHP (opcional se o arquivo tiver só PHP)

🧠 2. Variáveis e tipos de dados

<?php

$nome = "Maria";       // string (texto)

$idade = 25;           // inteiro

$peso = 60.5;          // número decimal (float)

$casada = false;       // booleano

echo "Nome: $nome\n";

echo "Idade: $idade\n";

echo "Peso: $peso kg\n";

echo "Casada: $casada\n"; // false aparece como vazio

?>

🧮 3. Operações matemáticas

<?php

$a = 10;

$b = 3;

echo "Soma: " . ($a + $b) . "\n";

echo "Subtração: " . ($a - $b) . "\n";

echo "Multiplicação: " . ($a * $b) . "\n";

echo "Divisão: " . ($a / $b) . "\n";

echo "Resto da divisão: " . ($a % $b) . "\n";

?>

💬 Observação:

  • O operador de concatenação em PHP é . (ponto), diferente do + em JavaScript.
  • Os operadores matemáticos são iguais nos dois.

🔀 4. Estruturas de decisão (if / else)

<?php

$idade = 17;

if ($idade >= 18) {

  echo "Maior de idade.";

} else {

  echo "Menor de idade.";

}

?>

🧩 Comparando com JavaScript:

  • Mesma estrutura de if e else.
  • Sem parênteses opcionais ou let/const (variáveis sempre com $).

🔁 5. Estruturas de repetição (loops)

🔹 Exemplo com while

<?php

$i = 1;

while ($i <= 5) {

  echo "Contando: $i\n";

  $i++;

}

?>

🔹 Exemplo com for

<?php

for ($i = 1; $i <= 5; $i++) {

  echo "Número $i\n";

}

?>

🔍 Comparando com JS:

  • A lógica é igual, apenas as variáveis têm $.

📦 6. Arrays (listas de valores)

<?php

$nomes = ["Ana", "Bruno", "Carlos"];

echo "Primeiro nome: " . $nomes[0] . "\n";

echo "Segundo nome: " . $nomes[1] . "\n";

// Adicionar novo elemento

$nomes[] = "Diana";

// Percorrer com foreach

foreach ($nomes as $n) {

  echo "Olá, $n!\n";

}

?>

🧩 Em JavaScript:

let nomes = ["Ana", "Bruno", "Carlos"];

nomes.push("Diana");

for (let n of nomes) {

  console.log("Olá, " + n + "!");

}
