Existem duas maneiras principais de fazer com que o echo do PHP não reconheça \n como uma quebra de linha, mas sim como o texto literal \n.

1. Use aspas simples (')

O PHP diferencia strings em aspas duplas (") e aspas simples ('). As aspas duplas interpretam sequências de escape como \n, \t, e variáveis, enquanto as aspas simples tratam o conteúdo literalmente.

Exemplo:

php

<?php

echo 'Aqui está um texto com \n que não será quebrado.';

?>

Saída:

Aqui está um texto com \n que não será quebrado.

2. Escape a barra invertida (\)

Se você precisar usar aspas duplas por algum motivo (por exemplo, para incluir variáveis na mesma string), você pode escapar a barra invertida com outra barra invertida (\\n) para que ela seja tratada como um caractere literal e não como uma sequência de escape.

Exemplo:

php

<?php

echo "Aqui está um texto com \\n que não será quebrado.";

?>

Saída:

Aqui está um texto com \n que não será quebrado.

Qual método usar?

  • Use aspas simples (') quando a string não contiver variáveis ou outras sequências de escape que você precisa que sejam interpretadas. É a forma mais simples e direta.
  • Use duas barras invertidas (\\) quando precisar usar aspas duplas para interpretar variáveis ou outras sequências de escape, mas quiser que \n seja tratado como texto literal.
